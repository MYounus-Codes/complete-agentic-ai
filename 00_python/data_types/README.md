# Data Types

This section explains the most common Python data types and how to use them safely.

## Covered topics

- Integers and floats
- Strings
- Booleans
- Lists, tuples, sets, and dictionaries
- Type conversion
- Type checking

## Files

- `data_types_intro.py` - core examples and type conversion
- `data_types_exercises.py` - short practice tasks
- `projects/basic/` - starter project
- `projects/medium/` - intermediate project
- `projects/advanced/` - extended project

## Goal

Understand how values are stored, how they behave, and when to choose one type over another.
