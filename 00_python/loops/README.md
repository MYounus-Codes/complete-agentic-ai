# Loops

This topic teaches `for` and `while` loops in a beginner-friendly order.

## What you will learn

- How to repeat code with `for` loops
- How to repeat code with `while` loops
- How to use `range`, `break`, and `continue`
- How to loop through lists and strings
- How to build small loop-based projects

## Learning order

1. Read `loops_intro.py`.
2. Run `loops_exercises.py` and change the numbers.
3. Build the basic project.
4. Try the medium project.
5. Attempt the advanced project after you understand the examples.

## Files

- `loops_intro.py` - simple examples with clear comments
- `loops_exercises.py` - short practice tasks
- `projects/basic/countdown_timer.py` - easy starter project
- `projects/medium/daily_menu_counter.py` - slightly more structured project
- `projects/advanced/study_tracker.py` - more complete project with multiple functions
