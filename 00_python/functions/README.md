# Functions

This section explains how to write reusable code with Python functions.

## Covered topics

- Defining functions
- Parameters and return values
- Default and keyword arguments
- Variable scope
- Lambda functions
- Recursion and simple higher-order functions

## Files

- `functions_intro.py` - function basics and examples
- `functions_exercises.py` - short exercises
- `projects/basic/` - starter project
- `projects/medium/` - intermediate project
- `projects/advanced/` - extended project

## Goal

Learn how to split problems into reusable pieces and keep code readable.
