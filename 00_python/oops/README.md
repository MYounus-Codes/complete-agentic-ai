# OOPs

This section introduces object-oriented programming with practical Python examples.

## Covered topics

- Classes and objects
- Instance, class, and static methods
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Files

- `oops_intro.py` - class and object basics
- `oops_exercises.py` - short exercises
- `projects/basic/` - starter project
- `projects/medium/` - intermediate project
- `projects/advanced/` - extended project

## Goal

Understand how to model real-world ideas with code that is easier to extend and reuse.

## Projects

- Basic: `projects/basic/pet_demo.py`
- Medium: `projects/medium/bank_account_system.py`
- Advanced: `projects/advanced/school_management_system.py`
